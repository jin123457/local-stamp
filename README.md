
# 🌿 사이사이

> **"내 취향을 따라 동네를 걷는 여정"**  
> **당신의 취향은 어느 골목에 머물고 있나요?**

📍 **배포 URL**: [사이사이 🌿 보러가기](https://saisai.lovable.app/)

---

## 🧭 프로젝트 개요

**사이사이**는 지역 상권 안에 있는 가게들을 **하나의 테마로 묶고**,  
사용자가 해당 그룹을 탐방하며 **스탬프를 적립하고 보상**을 받는  
**골목 상권 기반 취향 탐색 서비스**입니다.

우리는 이 서비스를 통해,  
단순한 소비를 넘어 **사람과 공간, 감정과 관계가 이어지는 연결의 여정**을 만듭니다.

---

## 💡 기획 의도

- 골목 상권이 **경험과 취향의 장소**로 다시 정의되기를 바람
- 관심사나 정체성을 기반으로 **사용자 주도의 탐색과 연결감**을 유도
- 지역 내 **가게 간 협업 구조와 상생 기반 네트워크** 조성
- 단골, 팬심, 정서적 공감 등을 바탕으로 한 **자기 발견형 소비 여정**

## 👥 핵심 페르소나

### 🧍‍♀️ Persona 1: 골목 탐방러 – 이수진 (28세, 마케터)
- **기본 정보**: 주말마다 핫플을 찾아 다니는 트렌드 러버
- **페인 포인트**: 
  - 매번 새로운 핫플을 찾기 위해 검색하는 시간이 부담
  - 방문한 곳들을 체계적으로 기록하고 관리하기 어려움
  - 혼자 가기엔 부담스럽고 친구들과 함께 할 재미있는 활동 부족
- **니즈/목표**:
  - 인증샷 + 기록 콘텐츠에 큰 흥미
  - '스탬프 챌린지'와 SNS 공유 요소를 즐김
  - 체계적인 탐방 루트와 성취감을 원함

### 🧍‍♂️ Persona 2: 생활밀착 소비자 – 김진우 (41세, 맞벌이 직장인)
- **기본 정보**: 반경 1km 이내의 상점에서 대부분의 소비를 함
- **페인 포인트**:
  - 늘 같은 가게만 이용하게 되어 새로운 발견이 없음
  - 할인 혜택을 놓치는 경우가 많아 아쉬움
  - 바쁜 일상 속에서 동네 탐방할 시간과 동기 부족
- **니즈/목표**:
  - 실용적인 할인 및 쿠폰에 관심
  - 반복 방문 시 보상 있는 구조에 높은 반응
  - 효율적인 동네 상권 활용을 원함

### 🧍 Persona 3: 지역 정서 사용자 – 장영순 (65세, 은퇴자)
- **기본 정보**: 오래 다닌 단골 가게와의 정서적 관계를 중시
- **페인 포인트**:
  - 복잡한 앱 사용법을 익히기 어려움
  - 새로운 가게에 가기엔 부담스럽고 용기가 필요함
  - 동네 변화를 따라가기 어려워 소외감을 느낌
- **니즈/목표**:
  - 복잡한 앱보다 직관적 UI 선호
  - '우리 동네'에 대한 애착이 강함
  - 익숙한 환경에서 점진적인 새로운 경험을 원함

### 🧍‍♂️ Persona 4: 팬 커뮤니티 탐방러 – 박성민 (34세, 스포츠 팬)
- **기본 정보**: 특정 팀(예: 수원 블루윙즈)을 열정적으로 응원
- **페인 포인트**:
  - 같은 팬들과 오프라인에서 만날 기회가 부족함
  - 팬심을 공유할 수 있는 장소와 콘텐츠 부족
  - 단순 응원을 넘어선 팬 활동의 다양성 부족
- **니즈/목표**:
  - 같은 팬이 운영하는 가게에 방문하며 정서 공유
  - 팬심을 기반으로 한 루트 콘텐츠를 즐김
  - 팬 정체성을 바탕으로 한 새로운 경험과 네트워킹을 원함

## 🏪 점주 페르소나

### 🧍‍♂️ Persona 5: 동네 창업가 – 정현우 (38세, 골목 카페 사장)

- **기본 정보**: 대전 중구 은행동에서 작은 카페 운영
- **라이프스타일**: 지역 커뮤니티 활성화에 관심 많음, 동네 네트워크 중요시
- **페인 포인트**:
  - 높은 마케팅 비용 대비 효과적인 고객 유입 어려움
  - 단골 고객 외 신규 고객 확보의 한계
  - 주변 상점과의 협력 기회 부족으로 혼자만의 영업
- **니즈/목표**:
  - 근처 상점과 상생할 수 있는 협력 모델 원함
  - 마케팅 예산이 적어도 참여 가능한 프로모션 원함
  - 신규 유입 고객을 늘릴 수 있는 수단 필요
- **서비스 이용 시나리오**:
  - 근처 서점, 베이커리, 꽃집과 쿠폰 그룹 구성
  - 고객에게 '은행동 5곳 스탬프 카드' 제공
  - 완성한 고객은 각 가게에서 소소한 혜택 제공
- **기회 포인트**:
  - 점주 간 협업형 프로모션 툴 제공
  - 자동 스탬프 인증 및 고객 관리 대시보드 제공

## 🧩 사용자 스토리

> 🙋‍♀️ **이수진**은 매주 새로운 핫플을 찾아 헤매던 중 '취향산책: 연남동 감성 카페 루트'를 발견합니다. 이제 검색 시간 없이도 체계적인 루트를 따라 주말마다 한 곳씩 방문하며 스탬프를 적립합니다. 각 가게에서의 특별한 순간들을 기록하고, 마지막 가게에서 받은 '디저트 쿠폰'을 친구와 함께 사용하며 인스타에 "🌿 #사이사이 #취향지도" 해시태그로 성취감을 공유합니다.

> 🙋‍♂️ **김진우**는 늘 같은 가게만 이용하던 일상에서 벗어나 '우리동네 실속 코스'에 참여합니다. 점심시간을 활용해 근처 새로운 가게들을 하나씩 방문하며, 각 가게에서 제공하는 실용적인 할인 혜택을 받습니다. 바쁜 와중에도 동네에 이런 좋은 가게들이 있었다는 것을 알게 되어 만족하며, 스탬프 완성 후 받은 종합 할인쿠폰으로 가족과 함께 식사를 즐깁니다.

> 🙋‍♂️ **박성민**은 '블루윙즈 팬존' 그룹에서 자신과 같은 팬심을 가진 가게들을 찾아다닙니다. 각 가게에서 사장님, 다른 팬들과 팀 이야기를 나누며 새로운 인맥을 쌓고, 한정판 스탬프를 인증하며 온라인 팬 커뮤니티에도 공유합니다. 이 여정은 단순한 소비가 아닌 "내 팬심을 걷는 길"이 되어 더욱 깊은 소속감과 즐거움을 선사합니다.

> 🧍 **장영순**은 복잡한 기능 없이 직관적인 화면으로 '우리동네 정겨운 코스'에 참여합니다. 40년 단골인 구멍가게부터 새로 생긴 작은 카페까지, 동네의 변화를 천천히 받아들이며 새로운 인연들을 만들어갑니다. 각 가게에서 따뜻한 대화를 나누고, 스탬프 완성 후 받은 소소한 선물에서 동네 공동체의 따뜻함을 다시 느낍니다.

> 🧍‍♂️ **정현우 사장님**은 인근의 서점, 베이커리와 함께 '은행동 감성 코스'를 구성하여 스탬프 카드를 운영합니다. 고객들은 골목 안에서 새로운 가게를 경험하고, 3곳 이상 방문 시 소소한 선물을 받아 갑니다. 정 사장님은 이를 통해 단골층 외 신규 고객과의 접점을 만들고, 주변 사장님들과의 네트워크도 확대하며, 무엇보다 적은 비용으로도 효과적인 마케팅 효과를 경험합니다.

## 🔧 MVP 기능 요약

| 분류 | 기능 | 설명 |
|------|------|------|
| 쿠폰 그룹 조회 | 홈에서 상권/테마 그룹 탐색 | 다양한 테마별 스탬프 루트 확인 |
| 가게 방문 | 그룹 내 가게 상세 보기 + 방문 버튼 | 각 가게 정보와 혜택 확인 |
| 스탬프 적립 | 버튼 클릭으로 방문 인증 | 간편한 방문 인증 (GPS/QR은 후속) |
| 쿠폰 진척 확인 | 스탬프 카드 UI로 여정 진행률 확인 | 시각적 진행률과 성취감 제공 |
| 보상 수령 | 스탬프 완성 후 보상 또는 쿠폰 제공 | 완주 시 특별 혜택 지급 |
| SNS 공유 | 앱 내 타 플랫폼 연동 공유 | 인스타그램, 카카오톡 등 간편 공유 |
| 마이페이지 | 완주 혜택 및 레벨 시스템 관리 | 개인 성취 기록과 레벨 확인 |

- **Backend**: Firebase Firestore (또는 Supabase)
- **Deploy**: Vercel
- **기획 도구**: Figma (UX 설계), Notion (프로젝트 관리)

## 🚶 사용 흐름 예시

1. 사용자가 '취향 루트: 수원 블루윙즈 팬존'을 선택
2. 참여 가게 리스트에서 첫 번째 가게 방문
3. 가게 상세 페이지에서 '방문 인증' 클릭 → 스탬프 적립
4. 스탬프 4개 완료 후, '응원 음료' 쿠폰 발급
5. 쿠폰 사용하며 팬들과 교류 인증 → SNS 공유
6. 마이페이지에서 완주 기록과 다음 레벨까지의 진행률 확인
